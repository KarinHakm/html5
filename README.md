# HTML5 Struktuuri Harjutus

## Valitud SaaS veebileht
**URL:** https://www.google.com/gmail/about/



## Struktuuri kirjeldus

Gmail avalehe struktuur koosneb järgmistest põhiplokkidest:

1. **Header** (Päis)
   - Logo vasakul
   - Navigatsioonimenüü keskel (For work, Features, Security)
   - Tegevusnupud paremal (Sign in, Create account)

2. **Hero sektsioon** (Peamine üleskutse)
   - Suur pealkiri (h1): "Secure, smart, and easy to use email"
   - Kirjeldus
   - Kaks Call-to-Action nuppu

3. **Gemini omaduste sektsioon**
   - Pealkiri (h2)
   - Kolm omaduskaarti (article) reas

4. **Turvalisuse sektsioon**
   - Pealkiri (h2)
   - Neli turvafunktsiooni (2x2 grid desktop'is)

5. **Produktiivsuse sektsioon**
   - Kolm tööriista/funktsiooni

6. **Mobiilirakenduse sektsioon**
   - Kolm mobiili-spetsiifilist funktsiooni

7. **Google Workspace integratsioon**
   - Info plokk

8. **FAQ sektsioon**
   - Korduma kippuvad küsimused

9. **Footer** (Jalus)
   - Navigatsioonilingid (Privacy, Terms, About, Help)
   - Copyright tekst (address element)

## Kasutatud semantilised elemendid
- `<header>` – lehe päis
- `<nav>` – navigatsiooniplokid (2 tk: header ja footer)
- `<main>` – peamine sisu
- `<section>` – sisuplokid teemade kaupa
- `<article>` – üksikud omadusekirjeldused
- `<footer>` – jalus
- `<address>` – kontaktinfo

## Ligipääsetavus
- ✅ `lang="et"` atribuut `<html>` elemendil
- ✅ Loogiline pealkirjade hierarhia (h1 → h2 → h3)
- ✅ Skip-link klaviatuurikasutajatele
- ✅ ARIA label'id navigatsiooniplokkel
- ✅ Responsiivne kujundus (320px–1920px)

## Tehniline info
- **Keeled:** HTML5, CSS3
- **Raamistikud:** Puuduvad (vanilla HTML/CSS)
- **JavaScript:** Ei kasutata
- **Valideeritud:** W3C HTML Validator
